# 蓄水池采样

* 问题：
  * 如何从一个集合(元素个数为`n`)中**随机**(等概率)地选取`m`个元素
  * 集合大小`n`要么未知，要么为大数
    * 无法一次性把集合中所有对象都加载至内存
    * **流式数据**
* 特例(`m=1`)
  * 解法
    * 以概率$\dfrac{1}{i}$($i \geq 1$)选择流式数据中第$i$个对象
  * 证明
    * 第$i$个对象最终被选择的概率 = 选择第$i$个对象的概率 * 其后面所有对象不被选择的概率
    * $p = \dfrac{1}{i} * (\dfrac{i}{i+1} * \dfrac{i+1}{i+2} * \cdots * \dfrac{n-1}{n}) = \dfrac{1}{n}$
