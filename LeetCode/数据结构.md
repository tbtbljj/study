* 哈希表(Hash)
  * 操作
    * Insert：`O(1)`
    * Delete：`O(1)`
    * Find：`O(1)`
  * `Hash Function`
    * 对于任意的key，得到一个**固定且无规律**的介于`0~capacity-1`的整数
    * 著名的哈希算法
      * MD5
      * SHA-1
      * SHA-2
  * `Collision`
    * Open Hashing
      * 经常使用
      * [开放散列](https://www.cs.usfca.edu/~galles/visualization/OpenHash.html)
    * Closed Hashing
      * [封闭散列](https://www.cs.usfca.edu/~galles/visualization/ClosedHash.html)
    * 再好的`hash function`也会存在**冲突**
  * `Rehashing`
    * 哈希表存储的元素太多（如超过容量的十分之一），将哈希表容量扩大一倍，并将所有的哈希值重新安排
```
int hashFunc(string key) {
    return md5(key) % HASH_TABLE_SIZE;
}

int hashFunc(string key) {
    int sum = 0;
    for (int i = 0; i < key.size(); ++i) {
        sum = sum * 31 + (int)(key[i]);
        sum = sum % HASH_TABLE_SIZE;
    }
    return sum;
}
```

  * 查询(find)
    * 时间复杂度：`O(1)`
    * 查询两个元素是否属于同一个集合
    * 递归 vs. 非递归
  * 合并(union)
    * 时间复杂度：`O(1)`
    * 合并两个集合
    * 集合`A`合并入集合`B`，只需修改集合`A`中代表元素的指向，即重新指向集合`B`中的代表元素
  * 派生操作
    * 查询某个元素所属集合的大小
    * 查询当前集合的个数
```
// 查询操作：无路径压缩
int find(int x) {
    if (father[x] == x) {
        return x;
    }
    return find(father[x]);
}
// 查询操作：有路径压缩O(1)
int find(int x) {
    if (father[x] == x) {
        return x;
    }
    return father[x] = find(father[x]);
}
// 查询操作：有路径压缩迭代版本

// 合并操作
void union(int x, int y) {
    int root_x = find(x);
    int root_y = find(y);
    if (root_x != root_y) {
        father[root_x] = root_y;
    }
}
// 并查集
class UnionFind {
public:
    int find(int x) {
        if (father[x] == x) {
            return x;
        }
        return father[x] = find(father[x]);
    }
    void union(int x, int y) {
        int root_x = find(x);
        int root_y = find(y);
        if (root_x != root_y) {
            father[root_x] = root_y;
        }
    }
private:
    vector<int> father;
};
```
* 堆(Heap)
  * 常用场景
    * 求集合元素最小值/最大值
  * 优先级队列(PriorityQueue)
    * [cppreference](https://en.cppreference.com/w/cpp/container/priority_queue)
    * 堆的一种实现方式
    * 头文件：`#include <queue>`
    * 与队列(queue)的不同
      * 可以自定义队列中元素的**优先级**
      * 优先级高的元素排在队列前面，优先出队
        * 优先级高先出
    * 基本操作（以下接口与栈相同）
      * `top()`：访问队头元素
      * `empty()`：检查队列是否为空
      * `size()`：返回队列中元素个数
      * `push(x)`：插入元素x并排序
      * `pop()`：弹出队头元素
    * 定义
      * priority_queue<Type, Container, Functional>
        * `Type`：数据类型
        * `Container`：容器类型
          * `STL`默认使用`vector`
        * `Functional`：比较方式
      * 使用**自定义数据类型**需要传入上述三个参数
      * 使用**基本数据类型**，只需要传入数据类型，默认是**大顶堆**
  * 例题
    * [LeetCode23 合并K个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/)
    * [LeetCode295 数据流的中位数](https://leetcode.cn/problems/find-median-from-data-stream/)
    * [LeetCode407 接雨水II](https://leetcode.cn/problems/trapping-rain-water-ii/)

* 逆波兰表达式
  * 定义：一种**后缀表达式**
    * 后缀：运算符写在运算数后面
    * 平时使用的算式是一种中缀表达式
      * 如，`(1 + 2) * (3 + 4)`
      * 对应的逆波兰表达式：`((1 2 + ) (3 4 + ) *)`
  * 优点
    * 去掉括号后表达式无歧义
      * 如，`1 2 + 3 4 + *`
    * 用**栈**操作运算
      * 遇到数字则入栈
      * 遇到算符则取出栈顶两个数字进行计算，并将结果入栈
